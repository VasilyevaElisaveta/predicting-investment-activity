### **Цель исследования**
Сравнение эффективности различных подходов машинного обучения и методов временных рядов для прогнозирования инвестиционной активности в субъектах РФ.

### **Структура исследования**
1. **Подготовка данных**  
   — Разделение на обучающую (до 2022) и тестовую (2023) выборки.  
   — Создание временных лагов и производных признаков.  

2. **Тестируемые модели**  
   — **Линейные модели:** Linear Regression, Ridge, Lasso, ElasticNet.  
   — **Методы временных рядов:** ARIMA, цепи Маркова, Linear Regression с лагами.  
   — **Нейросетевые архитектуры:** упрощённые реализации N-BEATS и Temporal Fusion Transformer (TFT).  
   — **Градиентные бустинги:** Gradient Boosting (с лагами и без), HistGradientBoosting, XGBoost, LightGBM, CatBoost.  

3. **Метрики оценки**  
   — MSE, RMSE, MAE, R².  

4. **Визуализация результатов**  
   — Сравнительные графики по всем метрикам.  
   — Scatter-plot предсказаний vs факта для топ-3 моделей.  

### **Ключевые результаты**
- **Наилучшие результаты** показали модели, учитывающие **временные лаги**.  
- **Gradient Boosting с лагами** и **Linear Regression с лагами** продемонстрировали наибольшую точность (R² > 0.96, низкие RMSE/MAE).  
- **Базовые линейные модели** без учёта временной динамики оказались недостаточно точными для задачи.  
- **Нейросетевые подходы** (N-BEATS, TFT) показали конкурентоспособные, но более затратные результаты.
